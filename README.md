# Installation
 - `git clone --recursive https://github.com/DesktopCoder1407/dotfiles.git ~/.dotfiles`
 - `cd ~/.dotfiles`
 - `./setup`

# Update
 - To update, simply run `./setup` again.

# Uninstall
 - To uninstall, run `./setup --uninstall`.
